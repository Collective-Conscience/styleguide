Images

All icons used in the Collective-Conscience app are from Google's Material Design collection. The Icon Font should be linked to in index.html, and icons should be inserted into html using "i" tags with the class "material-icons" (plus any modifying classes).

The resources for the icon font can be found at: https://material.io/icons/
